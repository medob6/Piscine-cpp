# 42 Piscine C++

Welcome to the **C++ Piscine** at **42 School**, an intensive coding bootcamp that provides hands-on exercises to build an in-depth understanding of **C++ programming**. Designed for individuals with a foundational knowledge of the **C language**, these modules focus on key concepts in modern object-oriented programming and data structures.

*All exercises are written in C++98, as mandated by the curriculum.*
